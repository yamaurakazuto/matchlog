  # matchlog

matchlog is the backend and frontend core of fanArena â€” a sports fan app that tracks real basketball player stats, game logs, and rankings.

This project is built to simulate how professional sports teams and leagues manage player statistics and fan-facing apps.

## Features

- Track game-by-game player stats (points, minutes, rebounds, assists)
- Aggregate season totals and averages
- Generate team rankings from normalized match logs
- Display data in a mobile-friendly fan app UI

## Tech Stack

- Frontend: React + Vite (generated from Figma design)
- Backend: Go (API server)
- Database: MySQL (normalized match log schema)
